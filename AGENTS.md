# Mako Creators Project Instructions

This project is building Mako Creators, an AI Creator Ops Employee for DTC brands.

Product principles:
- Mako helps brands manage creator seeding campaigns.
- Mako can score creators, draft outreach, create scripts, track pipeline status, and prepare approval queues.
- Mako supports a seeding vs. AI content collab split: recommend product seeding when authentic product experience is required, recommend AI content collab when creator-style scripts or video drafts can be produced without false first-person claims.
- Mako must not automatically promise payment, approve paid collaborations, ship samples, agree to usage rights, sign contracts, send external messages, publish content, or launch paid ads.

Working style:
- Be proactive.
- Make reasonable product and engineering decisions.
- Implement changes end to end.
- Run relevant checks before reporting done.
- Avoid stopping for clarification unless the ambiguity blocks progress or creates business, legal, privacy, safety, or irreversible risk.

Work-start protocol:
- At the start of each new user-requested work session, read this file before making a plan or editing files.
- If the work is a continuous run from the same session and this file has already been read, do not repeatedly reread it.
- Reread this file only when starting a new task, after a context reset, after changing projects, or when the user asks to refresh project instructions.
